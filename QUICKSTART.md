# Быстрый старт

## 1. Установка

```bash
cd bot
pip install -r requirements.txt
```

## 2. Настройка

Создайте файл `.env`:
```bash
cp .env.example .env
```

Отредактируйте `.env` и укажите ваши credentials от https://my.telegram.org/apps:
```
TG_API_ID=12345678
TG_API_HASH=your_api_hash_here
LOG_LEVEL=INFO
```

## 3. Первый запуск

```bash
python main.py
```

При первом запуске:
1. Введите номер телефона (в международном формате, например: +79991234567)
2. Введите код из Telegram
3. Если у вас включена двухфакторная аутентификация, введите пароль

После успешной авторизации сессия сохранится в `data/` и больше не потребуется авторизовываться.

## 4. Добавление в группу

1. Добавьте бота (ваш аккаунт) в группу/супергруппу
2. Выдайте права администратора с разрешением "Изменять информацию о группе"

## 5. Использование

### Переименование чата

**Способ 1:** Текст после команды
```
/rename Новое название
```

**Способ 2:** Ответ на сообщение
1. Найдите сообщение с нужным текстом
2. Ответьте на него командой `/rename`

### Установка фото чата

**Способ 1:** Прикрепите фото к команде
1. Напишите `/repic`
2. Прикрепите фото

**Способ 2:** Ответ на сообщение с фото
1. Найдите сообщение с нужным фото
2. Ответьте на него командой `/repic`

## Возможные проблемы

### "У бота нет прав администратора"
Выдайте боту права администратора в группе с разрешением изменять информацию.

### "Неправильные размеры фото"
Telegram требует квадратное фото. Используйте изображение с одинаковыми шириной и высотой.

### "Название чата слишком длинное"
Максимальная длина названия - 255 символов.

## Остановка бота

Нажмите `Ctrl+C` в терминале где запущен бот.
